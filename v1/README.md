# Liberty

**Liberty** is a minimalist web app that allows you to track assets, debts, and financial obligations through a simple, user-friendly interface. Itâ€™s designed to help you stay on top of your finances with ease.

## Features

* **Track Assets & Debts**: Effortlessly manage and visualize your financial obligations.
* **Income vs. Overhead**: Quickly compare monthly income against monthly overhead.
* **Net Worth Graph**: Easily set up and view your net worth over time.
* **Simple Setup**: Get started in no time with minimal configuration.

## Technical Stack

**Frontend**
* **Framework**: React
* **Language**: JavaScript

**Backend**
* **Database**: Supabase Database
* **Authentication**: Supabase Auth
